## Intro

Helper is a **One For All** modal interactive helper

## How to use
```tsx
import useHelper from '.';
const helper = useHelper();

helper.notification.warning("网络异常");

const modal = await helper.modal.open({
    render: () => <> hi </>,
    onBeforeConfirm: () => {
        console.log('onBeforeConfirm')
    },
    onConfirm(){
        console.log('onConfirm')
    }
});

```

## Design

**CoreContext**: 使用上下文管理状态，以便于全局调用，策略机制，以便于扩展

## Pseudo

```js
const helper = Helper
        .use('Model')
        .use('Confirm')
        .use('Notification')
        .init()
```
