## Intro

Helper is a **One For All** modal interactive helper

## How to use

[Reference](https://www.npmjs.com/package/aio-modal)

## Design

**CoreContext**: 使用上下文管理状态，以便于全局调用，策略机制，以便于扩展

## Pseudo

```js
const helper = Helper
        .use('Model')
        .use('Confirm')
        .use('Notification')
        .init()
```
