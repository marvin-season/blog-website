## Intro

ModelHelper is a **One For All** modal interactive helper

## How to use

[Reference](https://www.npmjs.com/package/aio-modal)

## Feature
**Context Manage** & **Declarative** & **Customize**

## Pseudo of definition

```js
const modelHelper = ModelHelper
        .connect('Model')
        .connect('Confirm')
        .connect('Notification')
        .new()
```
## Using case
```tsx
modelHelper.modal.open(<div>Your Modal</>)
```
